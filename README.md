# SkyBlock by prosto_sup4ik
Модпак для игры на скайблоке с модом Create, аддонами и еще кучей полезных вещей.

Для запуска создайте новую версию 1.21.1 neoforge(21.1.217 recommented) и один раз войдите в игру, чтобы у вас автоматичекси подгрузились необходимые файлы. Затем просто вставьте (с заменой) файлы в папку вашей сборки и вновь запустите игру. Наслаждайтесь!

Для создания пустого мира (скайблок) при создании мира выберите тип мира скайблок и нажмите "Настроить". В выпавшем меню выберите начальный остров "Island" - На нем будут все необходимые ресурсы для выживания.

Новые апдейты будут выходить по мере необходимости или при обновлении аддонов Create на нужную версию, таких как steam&rails, clockwork(valkirien skyes), big cannons.
