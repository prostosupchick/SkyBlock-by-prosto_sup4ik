# SkyBlock by prosto_sup4ik
Модпак для игры на скайблоке с модом Create, аддонами и еще кучей полезных вещей
Для запуска создайте новую версию и один раз войдите в игру, чтобы у вас автоматичекси подгрузились необходимые файлы. Затем просто вставьте (с заменой) файлы в вашу сборку и вновь запустит игру. Наслаждайтесь!
